Server Preparation\n```bash\n# Update system\nsudo apt update && sudo apt upgrade -y\n\n# Install Docker\ncurl -fsSL https://get.docker.com -o get-docker.sh\nsh get-docker.sh\nsudo usermod -aG docker $USER\n\n# Install Docker Compose\nsudo curl -L \"https://github.com/docker/compose/releases/download/v2.20.0/docker-compose-$(uname -s)-$(uname -m)\" -o /usr/local/bin/docker-compose\nsudo chmod +x /usr/local/bin/docker-compose\n\n# Install Node.js\ncurl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -\nsudo apt-get install -y nodejs\n```\n\n#### 2. Application Setup\n```bash\n# Create deployment directory\nsudo mkdir -p /opt/ai-suite\nsudo chown $USER:$USER /opt/ai-suite\n\n# Clone repository\ngit clone https://github.com/1234-ad/ai-productivity-suite.git /opt/ai-suite\ncd /opt/ai-suite\n\n# Setup environment\ncp .env.example .env\n# Configure production environment variables\n```\n\n#### 3. SSL Certificate (Let's Encrypt)\n```bash\n# Install Certbot\nsudo apt install certbot python3-certbot-nginx\n\n# Generate certificate\nsudo certbot --nginx -d yourdomain.com\n```\n\n#### 4. Deploy Application\n```bash\n# Build and start services\ndocker-compose -f docker-compose.prod.yml up -d\n\n# Run database migrations\ndocker-compose exec api npx prisma migrate deploy\n```\n\n##
